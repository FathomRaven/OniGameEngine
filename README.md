# OniGameEngine

Basic little game engine, based off of the tutorial series from Ather Omar.

Docs in github wiki:
https://github.com/FathomRaven/OniGameEngine/wiki

#### Libraries Used:

* SDL
* SDL_Image
* SDL_TTF
* SDL_Mixer

#### Compiling

Run make, uses clang++

Executable must be in debug in order to find assets
